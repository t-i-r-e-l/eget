# Пример файла со списком доменов для маршрутизации
# Этот файл используется в конфигурации routing с типом "list"
#
# Формат:
# - Каждая строка = один домен
# - Строки, начинающиеся с #, являются комментариями
# - Пустые строки игнорируются
# - Домены, начинающиеся с точки (например, .google.com), считаются суффиксами
# - Поддерживаются IDN домены (кириллица и другие не-ASCII символы)
# - Комментарии в конце строки также поддерживаются

# Точные совпадения доменов
example.com
test.example.org
blocked-site.com

# Суффиксы доменов (все поддомены будут совпадать)
# Домены, начинающиеся с точки, применяются ко всем поддоменам
.google.com
.facebook.com
.twitter.com

# Еще примеры точных доменов
youtube.com
github.com
stackoverflow.com

# IDN домены (кириллица) - поддерживаются автоматически
# пример.рф
# тест.москва

# Комментарии в конце строки также работают
wikipedia.org  # Википедия
reddit.com  # Reddit

# Пустые строки игнорируются


# Больше примеров
discord.com
telegram.org
signal.org
